Key Components:
==============

1. **MainApp**: Root widget that sets up MaterialApp with dark theme
2. **CompareBody**: Main stateful widget managing the entire UI
3. **Product Cards**: Two identical card structures for product input
4. **VS Divider**: Decorative divider between product cards
5. **Action Buttons**: Compare and Clear buttons
6. **Result Card**: Conditional widget showing comparison results

State Management:
================
- TextEditingController for each input field (6 total)
- String variables for unit selection (2 total)
- ComparisonResult? for storing comparison results
- List<String> for available units

Theme Configuration:
==================
- Dark theme with custom colors
- Gradient backgrounds
- Custom input decoration
- Elevated button styling
- Card styling with borders and shadows
